External Review

This endpoint is only available for pro and ultra tiers.

URL path: /external_reviews/

Example response

{
    "id": 50331,
    "game": 7621,
    "source": 30,
    "score": 40,
    "url": "http://www.impulsegamer.com/rime-switch-review/"
}

Fields

Name Type Mandatory Comment
id unsigned 64-bit integer +
game unsigned 64-bit integer + ID of a Game record
source unsigned 64-bit integer + ID of an External Review Source
url string +
score integer The review score out of 100
